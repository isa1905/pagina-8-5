MATEMÁTICAS

1.¿QUÉ SON PRODUCTOS NOTABLES?

>Los productos notables son simplemente multiplicaciones especiales entre expresiones algebraicas las cuales sobresalen de las demás multiplicaciones por su frecuente aparición en matemáticas. De ahí el nombre producto, que hace referencia a "multiplicación" y notable, que hace referencia a su "destacada" aparición.

Así bien, una vez aprendido dichos productos notables, no habrá necesidad de comprobar dicha multiplicación mecánicamente, es decir, solo debemos seguir las reglas aprendidas con anterioridad que caracterizan a cada producto notable.

nos referimos a una fórmula algebraica específica que involucra el uso de la diferencia de cuadrados. Si tienes dos binomios, por ejemplo (a + b) (a + b) (a + b) y (a−b) (a - b) (a−b), el producto de su suma por su diferencia se calcula de la siguiente manera:

3.PRODUCTO DE DOS BINOMIOS DE LA FORMA (a + b) (a + c)

Para multiplicar los binomios (a+b) y (a+c), utilizamos la propiedad distributiva (también conocida como el método FOIL para binomios). Aquí está el proceso paso a paso:


1. Expresa el producto:

(a +b)(a +c)(a + b) (a + c)(a +b)(a +c)

2. Aplica la propiedad distributiva: Multiplicamos cada término del primer binomio por cada término del segundo binomio:
(a+b) ( a+ c)=a( a +c) +b( a +c)(a + b) (a + c) = a (a + c) + b (a + c) (a+b)(a +c)=a(a +c)+b(a +c)

3. Desarrolla cada producto: a(a+c)=a2+aca(a + c) = a^2 + aca(a+c)=a2+ac b(a+c)=ab+bcb(a + c) = ab + bcb(a+c)=ab+bc

4. Suma los resultados: a2+ac+ab+bca^2 + ac + ab + bca2+ac+ab+bc
Por lo tanto, el producto de los binomios (a+b) (a + b)(a+b) y (a+c)(a + c)(a+c) es:

a2+ab+ac+bca^2 + ab + ac + bca2+ab+ac+bc

4.CUADRADO DE UN BINOMIO SUMA Y RESTA

Suma: Un binomio al cuadrado (suma) es igual es igual al cuadrado del primer término, más el doble producto del primero por el segundo más el cuadrado segundo. (a + b)2 = a2 + 2 · a · b + b2
