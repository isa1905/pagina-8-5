CIRCUITOS ELÉCTRICOS:

¿qué es un circuito?

Un circuito es un conjunto de componentes electrónicos interconectados que permiten el flujo de corriente eléctrica para realizar una función específica, como amplificar una señal, controlar un dispositivo o procesar información.

¿con qué esta formado?

Un circuito eléctrico está formado por componentes electrónicos como resistencias, condensadores, inductores, diodos, transistores, entre otros, conectados entre sí.

¿Cuáles son los tipos de circuitos?

- Circuitos abiertos: No permiten el flujo de corriente porque tienen un interruptor abierto o un componente defectuoso.
- Circuitos cerrados: Permiten el flujo de corriente porque todos los componentes están conectados correctamente.
- Circuitos serie: Los componentes están conectados uno después del otro, formando una línea.
- Circuitos paralelos: Los componentes están conectados entre sí, formando múltiples caminos para la corriente.

¿para qué sirve un circuito?

- Controlar el flujo de corriente eléctrica.
- Procesar señales.
- Realizar funciones lógicas.
- Proporcionar energía.
- Medir magnitudes.
- Conectar dispositivos.
- Proteger dispositivos.
